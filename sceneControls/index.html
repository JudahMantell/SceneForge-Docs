    Scene Controls

    The Scene Toolbar

    On the desktop view there is a toolbar at the top of the window.

    The toolbar displays the project name, as well as several functions for the project. The options available are as follows:

    ├─ SceneForge Menu Bar
│  └─ Project_Name         --- Click here to open the project directory.
│  └─ File
│       └─ Save Project    --- Saves the Project
│       └─ Refresh Assets  --- Reloads the Asset Database*
│       └─ Project Output  --- Displays the raw json data of the scene
│  └─ Settings             --- Opens a Settings window
│  └─ Scripting            --- Opens a Scripting window (Coming Soon!)*
│  └─ Camera               --- Opens a Camera Options window*
│  └─ Exit                 --- Quits SceneForge

    A few notable options:

    • Refresh Assets reloads the database of objects and textures, allowing for importing new assets while the application is running. This will cause the VR display to fade to black for a moment, allowing all the new assets to be loaded in without dropping frames.
    • Scripting is a feature coming in future releases.
    • Camera
      • After the scene is finished loading, there is an option for `Camera Smoothing. This adds smoothing to the desktop view camera. It has no affect in VR, but might affect performance.
      • When the camera is activated, there is an option for Mobile Cam. This enables the mobile app functionality that isn't available just yet.

    Navigating the Scene

    To move around the scene, you have three options.

    1. You can physically walk around your designated play area.
    2. Hold both grip buttons (or grab on Valve Index) and and manipulate the scene around you.
    3. Hold the left trigger to show a teleport indicator. Release the trigger to teleport to the chosen spot.

    Manipulating Objects

    Your right controller has a pointer that can interact with UI and objects. To interact with an object, click the right trigger while pointing at at either a UI panel or object in the scene. When you click an object, it becomes highlighted. Click the object again to unhighlight it.

    A selected object gets highlighted in orange. To move an object or UI panel around the scene, make sure it's selected, then point at the object, and hold down the grip button.

    Use the Right Joystick or Touchpad push it forward or back (up or down on the touchpad/joystick), or scale it up and down (left and right on the touchpad/joystick).

    To toggle rotate mode (as indicated by the small icon on the right controller), press the (B) button on your right controller. This button varies depending on what controllers you are using, but they are largely the same. Now you can use the touchpad/joystick to rotate the object along different axis.

    The Properties Menu

    When an object is selected, a properties panel will popup in front of you. You can move this menu around using the grip button/pointer, or reaching out and actually grabbing it.

    Every object, when selected (except backdrops), brings up this default toolbar.
    These functions delete the object, reset the rotation of the object, or toggle interactability. (Whether or not the object can be grabbed with the grip button. This is not recommended for larger objects.)

    Below this mini-toolbar, are options specific to the type of object. In this example, a light is selected.

    Backdrop Selector Panel

    If you select a backdrop object with the pointer, it will highlight and the properties panel will change to a backdrop selector panel. Here you can choose from a grid of textures to apply to the backdrop. These textures are taken from the Textures folder in your project. See Getting Started page for more info about custom assets.

    The Scene Menu

    Your Left Controller can toggle a "Scene Menu" that can bring up other menus.

    From top to bottom, each button changes what menu is displayed to the right (see below).

    For a detailed overview of each menu, see section below.

    • Objects -- This menu displays a grid of available objects including standard assets and imported assets. Select an item to spawn it into the scene.
    • Lighting -- This menu has many functions to change the lighting of the scene (see below).
    • Camera -- This menu brings up global settings for all cameras in the scene. This includes resolution settings, and many different post processing effects.
    • Settings -- This menu has many functions relating to the opened project and Sceneforge as a whole (see below).

    Object Spawn Menu

    This panel will display a list of available objects to spawn into the scene. This list includes custom imported objects, and objects that are included with the software. In the future there will also be options to spawn in effects (weather, etc).

    To spawn in an object, just point and click on one of the tiles. The object will spawn in front of you, and will be added to the scene hierarchy

    You can also spawn lights and cameras, which each have their own unique properties. See Camera page for more info.

    Lighting Menu Panel

    This panel has two options: * A list of all available skyboxes for the scene. Select a skybox to assign it to the scene. The ambient lighting of the scene may also change to reflect the skybox. * Options for dymanic time of day and weather settings. These settings overrride any custom skybox options and are only available for the "Basic" and "Pro" plans.

    Settings Menu Panel

    This panel has two options for either project management or global editor settings. These options are self-explanatory.
